1. 將 Loading1.gif 放到與 index.html 同一層資料夾（檔名需為 Loading1.gif）。
2. 用瀏覽器開啟 index.html。
3. 首頁填寫姓名與電子信箱、勾選隱私權，按「開始能力值調查」。
4. 依序完成 8 題（前 7 題計分，第 8 題僅記錄期待）。
5. 頁面會顯示：
   - 你的姓名＋結果稱號
   - 一段人格描述
   - 一段適合的產業傾向說明
   - 三則推薦課程方向
   - 下方按鈕：重新測驗、分享結果、前往「產業新尖兵計畫」網頁。
